# LLM Sandpit

Experiments with LLMs and the like

## Huggingface Sandpit Usage

Using a Python virtual environment:

### Create/Activate

```sh
cd huggingface/sandpit
virtualenv venv
. venv/bin/activate
```

### Install Dependencies

```sh
pip install -r requirements.txt
```

### Deactivate

```sh
deactivate
```
